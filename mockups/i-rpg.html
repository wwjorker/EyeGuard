// 8-BIT RPG QUEST

把护眼变成 RPG 任务。你是一个像素战士，HP = 健康分，XP = 连续天数。

休息 = 喝补血药水。完成番茄循环 = 击败 boss。每天连击 = 升级解锁新装备。

每个提醒都用 RPG 对话框样式（守护者 Iris 是你的旁白）。

很有 viral 潜力 — 喜欢 Stardew Valley / Undertale / 像素游戏的群体会爱死。

工作量：4-5 轮（要做角色 sprite + 对话系统 + RPG 文案重写 + 升级等级 + 装备系统）。是这套里最大的工程。

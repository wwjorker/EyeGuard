// 整套统一的样子

同一个窗户、同一个色彩体系、同一个植物隐喻贯穿四个页面。点上面 tab 切换感受一下：

📍 timer 页

  • 窗外天空 + 窗台上的植物
  • 计时器是窗户上的卡片
  • 四个统计藏在窗台底部

📊 stats 页

  • 背景换成纸黄（像是翻开的"花园日记本"）
  • 4 个统计卡 = 种子袋
  • 小时分布 = 一排逐渐长高的草
  • 应用分布 = 圆形的"花圃"
  • 7 天柱图 = 一排花茎

🍅 pomodoro 页

  • 4 个番茄挂在藤蔓上
  • 已完成 = 红色摘下；当前 = 红色脉动；未来 = 绿色未熟
  • "harvest ahead in 2 tomatoes" 真的有种期待感

⚙️ settings 页

  • 每个分组 = 一块"花圃"
  • 滑块 = 木质藤蔓 + 花苞滑钮
  • 开关 = 闭合/绽放的花苞
  • 分段 = 花瓣选择

通知窗口 / 休息窗口我这没画但思路也一致：通知是"飘过的小种子"，休息是"看向窗外的安静风景"，结束时窗台多一片新叶。

⏳ 工作量估算

  • 第 1 轮：timer 页 + 主窗口框架 + 色彩 / 字体体系
  • 第 2 轮：stats 页（4 种数据可视化重画）
  • 第 3 轮：pomodoro 页（番茄藤动画 + 花圃化设置）
  • 第 4 轮：settings 页 + 通知窗口 + 休息窗口
  • 合计 3-4 轮。每轮我都先 build 验证不破坏功能。

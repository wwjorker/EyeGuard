# EyeGuard 设计规范 (Design System) — Type + Rings

> 本文档由 SRS v4.0 第 4 章提取而成，作为后续每个 Phase 实施时的视觉/交互约束。

## 1. 设计理念

视觉风格：**Type + Rings** — 超大渐变淡出字体 + 多色指标小环横排，高端表盘感与极简排版的融合。

核心原则：
- **字体即主角**：计时器数字超大加粗（56px+），使用 gradient fade（从白到半透明）创造深度层次。
- **多色环形指标**：底部横排 4 个小圆环，每个代表一个维度（绿=屏幕时间、紫=休息、黄=评分、粉=连续时长），彼此用淡线分隔。
- **暗色留白**：近纯黑背景搭配大量留白，让核心信息"呼吸"，视觉不拥挤。
- **圆角边框框架**：整个内容区外围一圈极淡的圆角边框（`rgba(255,255,255,0.04)`），营造精致感。

## 3. 字体

全局只使用 **Inter**（Google Fonts）。

| 用途 | 大小 | 样式 |
|---|---|---|
| 主计时器数字 | 56px / weight 800 / letter-spacing -3px | 渐变淡出（背景渐变 + 文本剪切） |
| 品牌标识 | 10px / weight 400 / letter-spacing 3px / 大写 | `rgba(255,255,255,0.2)` |
| 时间下方标签 | 11px / letter-spacing 1px | `rgba(255,255,255,0.25)` |
| 环内数字 | 14px / weight 700 | 白色 |
| 环下标签 | 8px / 大写 / letter-spacing 1.5px | `rgba(255,255,255,0.25)` |
| 设置/统计页标题 | 18-22px / weight 600 | 白色 |
| 正文 | 14px / weight 400 | 正常颜色 |
| 数据数字 | 20-24px / weight 700 | 白色 |
| 徽章文字 | 10px | 对应颜色 |

主计时器渐变：
```css
background: linear-gradient(180deg, #fff 40%, rgba(255,255,255,0.4));
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```

│  │                                    │  │
│  │           14:32                    │  │
│  │       minutes remaining            │  │
│  │                                    │  │
│  │  (○3:20)─(○4/6)─(○85)─(○42m)       │  │

- 外层：主背景 `#09090B`
- 内层：`inset: 12px` 的圆角边框（20px 圆角，`rgba(255,255,255,0.04)`）
- 顶栏导航：4 个 dot 或顶部标签（timer / stats / pomodoro / settings）

## 5. 核心组件规范

### 超大计时器（Hero Timer）
- 字号 56px，weight 800，letter-spacing -3px。
- 渐变淡出顶纯白 → 底 40% 透明白。
- 下方 "minutes remaining" (11px / `rgba(255,255,255,0.25)` / letter-spacing 1px)。
- 暂停态：整体 30% 透明度。
- 即将到期（<2分钟）：颜色渐变切到 `#F59E0B`。

### 四环指标行（Metric Rings Row）
- 4 个 SVG 圆环，60px 直径，3px 描边。
- 底环：`rgba(255,255,255,0.05)`。
- 进度环：四色 stroke-dasharray 控制。
- 环心数字：14px 粗体白色。
- 环下标签：8px / 大写 / letter-spacing 1.5px / `rgba(255,255,255,0.25)`。
- 环间用 `linear-gradient(90deg, transparent, rgba(255,255,255,0.06), transparent)` 连接。
- 四色含义：绿=屏幕时间、紫=休息次数、黄=健康评分、粉=最长连续。

### 状态徽章（Status Badge）
- 圆角药丸（border-radius 20px）。
- 背景 `rgba(52,211,153,0.1)`，边框 `rgba(52,211,153,0.15)`，文字 `#34D399`。
- 文案："active" / "paused" / "break" / "20-20-20" / "pomodoro"。
- 暂停时灰色系。

### 浮窗提醒（中提醒 Toast）
- 右下角滑入：translateX(120%)→0，300ms ease-out。
- 背景 `#18181B`，圆角 16px，边框 `rgba(255,255,255,0.06)`。
- 左侧 3px 绿色竖条 `#34D399`。
- 内容：lucide 图标 + 标题 + 连续使用时长。
- 按钮：「开始休息」(填充 #34D399 / 文字 #09090B) / 「稍后」(ghost 边框 `rgba(255,255,255,0.1)`)。
- 8 秒自动滑出。

### 全屏休息遮罩（强提醒）
- 全屏 `rgba(9,9,11,0.94)` + `backdrop-filter: blur(24px)`。
- 中央 80px 倒计时（weight 200 超细体，渐变淡出）。
- 冒号 `rgba(52,211,153,0.6)`。
- 极淡圆形轨道 `rgba(255,255,255,0.03)`，绿色光点 `#34D399` 沿轨匀速旋转 8s linear infinite，`box-shadow: 0 0 10px`。
- 下方护眼小操引导。
- 底部 "跳过" ghost 按钮。

### 设置页面
- 分组卡片 `#18181B` / 16px 圆角，组内分隔线 `rgba(255,255,255,0.06)`。
- 滑块：轨道 `#27272A`，已选区 + 圆点 `#34D399`。
- 开关：关 `#3F3F46` / 开 `#34D399`。
- 输入框：背景 `#09090B`，边框 `rgba(255,255,255,0.06)`，聚焦绿边。

### 图表（Recharts）
- 柱状图：四色交替填充，hover 亮度提升。
- 环形图：四色体系。
- 折线图：绿色线 + 绿色半透明面积。
- 坐标轴文字 `rgba(255,255,255,0.25)` 11px。
- 网格线 `rgba(255,255,255,0.04)`。
- 背景透明。

## 6. 动画规范

| 动画 | 触发 | 参数 |
|---|---|---|
| 数字渐变 | 计时器每秒更新 | 平滑递减、无闪烁 |
| 环形进度 | 数据变化 | stroke-dashoffset 800ms ease |
| 轨道光点 | 全屏休息 | 8s linear infinite 旋转 |
| 浮窗滑入 | 中提醒 | translateX 120%→0, 300ms ease-out |
| 浮窗滑出 | 关闭/超时 | translateX 0→120%, 200ms ease-in |
| 全屏淡入 | 强提醒 | opacity 0→1, 500ms ease |
| 页面切换 | 导航点击 | opacity 0→1, 200ms ease |
| 数字滚动 | 统计变化 | 旧→新, 400ms |
| 按钮交互 | hover/active | scale(1.02) / scale(0.98), 150ms |
| 暂停过渡 | 暂停/恢复 | 透明度 100%↔30%, 300ms ease |

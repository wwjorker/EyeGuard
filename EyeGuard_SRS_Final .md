# EyeGuard 软件需求规格说明书 (SRS) v4.0

## 1. 项目愿景

EyeGuard 是一款智能护眼助手桌面应用。它通过感知用户行为、分级提醒、应用级用眼统计和数据洞察，将护眼从"被动打断"变为"主动习惯"。

**一句话定位：** 不是又一个 20-20-20 闹钟，而是你的个人用眼健康助手。

**差异化优势：**
- 应用级用眼统计（知道你在哪个软件上花了多久）
- 数据可视化洞察（趋势图、对比分析）
- 渐进式分级提醒（不会让你想卸载）
- Type + Rings 视觉风格（超大渐变字体+多色指标环，高端辨识度）
- 中英双语支持

## 3. 核心功能模块

### A. 智能感知模块 (Smart Sensing)

**[F-A01] 用户状态检测**
- 通过全局键盘/鼠标事件监听判断用户是否在使用电脑
- 触摸屏操作在系统层映射为鼠标事件，无需额外处理
- 闲置阈值可自定义（默认 5 分钟），超时自动暂停计时
- 用户回归后自动恢复计时，无需手动操作

**[F-A02] 智能延期 (Smart Snooze)**
- 自动检测当前是否处于全屏状态（游戏、视频、演示文稿等）
- 全屏状态下静默推迟提醒（默认推迟 10 分钟）
- 安全上限：最多连续推迟 3 次（即最多额外 30 分钟），之后强制提醒
- 推迟次数和单次推迟时长均可在设置中自定义

**[F-A03] 应用足迹 (App Footprint)**
- 通过系统 API 获取当前前台窗口的进程名和窗口标题
- Windows 使用 `GetForegroundWindow` + `GetWindowText` API
- 每 5 秒采样一次前台应用，记录到本地数据库
- 此功能默认开启，用户可随时关闭
- 用户可一键清除全部历史记录

---

### B. 分级提醒系统 (Tiered Alerts)

提供三档模式，用户可在设置中选择默认模式（默认为"中提醒"）：

**轻提醒 (Light)**
- 系统原生通知气泡
- 仅提示文字，不打断工作流
- 适合：会议中、轻度使用

**中提醒 (Medium) — 默认**
- 屏幕右下角弹出浮窗卡片
- 显示倒计时 + 本次连续使用时长
- 可点击"开始休息"进入全屏休息，或点击"稍后"延迟 5 分钟
- 适合：日常工作

**强提醒 (Hard)**
- 全屏半透明磨砂遮罩覆盖所有屏幕
- 显示 20 秒倒计时（可自定义时长）
- 默认可点击"跳过"关闭；开启严格模式后必须等待倒计时结束
- 休息期间展示护眼小操引导（见 C 模块）
- 适合：长时间高强度使用、希望强制执行的用户

**通用提醒参数（均可自定义）：**
- 工作间隔：默认 20 分钟（范围 10~120 分钟）
- 休息时长：默认 20 秒（范围 10~300 秒）
- 提醒声音：可选开关 + 音量调节

---

### C. 健康互动模块 (Interactive Health)

**[F-C01] 护眼小操**
- 强提醒的全屏休息界面中，随机展示眼部运动引导
- 内容包括：眼球上下左右转动、远眺提示、用力闭眼放松、眨眼练习
- 以文字 + 简单动画（CSS 动画）形式呈现，不依赖外部资源

**[F-C02] 饮水提醒**
- 休息结束时，以一定概率（默认 50%）显示饮水提示
- 可在设置中关闭

**[F-C03] 姿势提醒**
- 每隔 N 次休息提醒（默认每 3 次），附带一条姿势建议
- 内容包括：坐直、调整屏幕距离、放松肩颈
- 可在设置中关闭

---

### D. 数据统计模块 (Analytics Dashboard)

**[F-D01] 今日概览**
- 今日累计用眼时间
- 已完成休息次数 / 跳过次数
- 当前连续使用时长
- 最长一次连续使用时长

**[F-D02] 应用分布**
- 饼图/环形图展示各应用占用时间比例
- 按类别分组：生产力工具、浏览器、娱乐、通讯、其他
- 用户可自定义应用分类规则

**[F-D03] 趋势分析**
- 每日/每周用眼时长柱状图
- 对比指标："本周 vs 上周"用眼变化百分比
- 休息完成率趋势线

**[F-D04] 数据管理**
- 支持导出数据为 JSON 或 CSV 格式
- 支持一键清除全部统计数据
- 数据保留周期：默认 90 天，可自定义

---

### E. 效率增强模块 (Productivity)

**[F-E01] 番茄钟模式**
- 可选开启，替代默认的自由计时模式
- 默认 25 分钟工作 + 5 分钟休息
- 每 4 个番茄钟后触发一次长休息（默认 15 分钟）
- 工作时长、短休息、长休息、循环次数均可自定义
- 休息阶段复用提醒系统的全屏休息界面

---

### F. 系统集成模块 (System Integration)

**[F-F01] 系统托盘**
- 应用启动后常驻系统托盘，主窗口可关闭（最小化到托盘）
- 托盘图标显示当前状态（正常计时 / 已暂停 / 休息中）
- 左键单击：打开/聚焦主窗口
- 右键菜单：
  - 暂停 / 恢复计时
  - 立即休息
  - 打开设置
  - 退出应用

**[F-F02] 全局快捷键**
- `Ctrl+Shift+P`：暂停/恢复计时
- `Ctrl+Shift+B`：立即开始休息
- 快捷键可在设置中自定义或禁用

**[F-F03] 开机自启**
- 可选开关，默认关闭
- 通过系统注册表或启动目录实现

**[F-F04] 勿扰白名单**
- 用户可指定应用列表，当这些应用处于前台时不弹提醒
- 典型场景：演示软件、会议软件（如 Zoom、Teams）
- 白名单内的应用仍会被计入用眼统计

### G. 界面与主题 (UI & Theming)

**[F-G01] 主题切换**
- 深色模式为默认主题（Cyberpunk Dark）
- 支持浅色模式切换（浅色模式采用同一套设计语言，绿色主色调不变）

## 4. 设计规范 (Design System) — Type + Rings

### 4.1 设计理念

视觉风格：**Type + Rings** — 超大渐变淡出字体 + 多色指标小环横排，高端表盘感与极简排版的融合。

核心原则：
- **字体即主角**：计时器数字超大加粗（56px+），使用 gradient fade（从白到半透明）创造深度层次，一眼记住
- **多色环形指标**：底部横排 4 个小圆环，每个代表一个维度（绿=屏幕时间、紫=休息、黄=评分、粉=连续时长），彼此用淡线分隔
- **暗色留白**：近纯黑背景搭配大量留白，让核心信息"呼吸"，视觉不拥挤
- **圆角边框框架**：整个内容区外围一圈极淡的圆角边框（`rgba(255,255,255,0.04)`），营造精致感

### 4.4 布局结构

**主窗口（约 420×580px 或可调）：**
```
┌──────────────────────────────────────────┐
│  ┌────────────────────────────────────┐  │
│  │  eyeguard              [active] ●  │  │  ← 顶栏：品牌 + 状态徽章 + 导航dots
│  │                                    │  │
│  │                                    │  │
│  │           14:32                    │  │  ← 超大渐变淡出时间
│  │       minutes remaining            │  │  ← 淡色标签
│  │                                    │  │
│  │                                    │  │
│  │  (○3:20) ─── (○4/6) ─── (○85)──(○42m)│  ← 底部四环横排
│  │  screen    breaks    score  streak │  │
│  └────────────────────────────────────┘  │
└──────────────────────────────────────────┘
```

- 外层：主背景 `#09090B`
- 内层：内容区域被一个 `inset: 12px` 的圆角边框（20px 圆角，`rgba(255,255,255,0.04)` 边框）包围
- 导航方式：顶栏右侧用小圆点指示器（4 个 dot），或者顶部标签切换（timer / stats / pomodoro / settings），不使用侧边栏
- 这种布局更紧凑，适合桌面小窗口常驻

### 4.5 核心组件规范

**超大计时器（Hero Timer）：**
- 字号 56px，font-weight 800，letter-spacing -3px
- 颜色使用 CSS 渐变淡出效果（顶部纯白 → 底部 40% 透明白），形成"浮出"的视觉层次
- 下方标签 "minutes remaining"（11px，`rgba(255,255,255,0.25)`，letter-spacing 1px）
- 工作中：正常显示
- 暂停态：数字整体降为 30% 透明度
- 即将到期（<2分钟）：数字颜色渐变从白色切换为 `#F59E0B`（琥珀色）

**四环指标行（Metric Rings Row）：**
- 底部水平排列 4 个小圆环，每个 60px 直径，3px 描边
- 底环（track）：`rgba(255,255,255,0.05)`
- 进度环（fill）：各自颜色（绿/紫/黄/粉），通过 SVG stroke-dasharray 控制进度
- 环心数字：14px 粗体白色
- 环下标签：8px，大写，`rgba(255,255,255,0.25)`，letter-spacing 1.5px
- 环与环之间用淡色横线 `linear-gradient(90deg, transparent, rgba(255,255,255,0.06), transparent)` 连接
- 四个指标：
  - 绿色 `#34D399`：屏幕时间（如 3:20）
  - 紫色 `#6366F1`：休息次数（如 4/6）
  - 黄色 `#F59E0B`：健康评分（如 85）
  - 粉色 `#EC4899`：最长连续（如 42m）

**状态徽章（Status Badge）：**
- 右上角，圆角药丸形（border-radius 20px）
- 背景 `rgba(52,211,153,0.1)`，边框 `rgba(52,211,153,0.15)`，文字 `#34D399`
- 显示当前模式："active" / "paused" / "break" / "20-20-20" / "pomodoro"
- 暂停时徽章切换为灰色系

**浮窗提醒（中提醒 Toast）：**
- 从屏幕右下角滑入（translateX(120%) → translateX(0)，300ms ease-out）
- 背景：`#18181B`，圆角 16px，边框 `rgba(255,255,255,0.06)`
- 左侧竖条：3px 宽，`#34D399`
- 内容：lucide 图标 + 标题 + 连续使用时长
- 按钮："开始休息"（填充 `#34D399`，文字 `#09090B`）/ "稍后"（ghost，边框 `rgba(255,255,255,0.1)`）
- 8秒后自动滑出消失

**全屏休息遮罩（强提醒）：**
- 全屏覆盖，背景 `rgba(9, 9, 11, 0.94)` + `backdrop-filter: blur(24px)`
- 中央超大倒计时数字（80px，font-weight 200 超细体，白色渐变淡出效果）
- 冒号用绿色 `rgba(52,211,153,0.6)` 点缀
- 数字外围一个极淡的圆形轨道（`rgba(255,255,255,0.03)`），轨道上有一个缓慢移动的绿色光点 `#34D399`（带 `box-shadow: 0 0 10px`）
- 倒计时下方：护眼小操引导区域
- 底部 "跳过" ghost 按钮（边框 `rgba(255,255,255,0.1)`，hover 填充 `rgba(255,255,255,0.05)`）

**设置页面：**
- 顶部标签切换到 "settings" 页
- 分组卡片布局，每组标题带 lucide 图标
- 卡片背景 `#18181B`，圆角 16px，内部分隔 `rgba(255,255,255,0.06)` 细线
- 滑块：轨道 `#27272A`，滑块圆点和已选区域为绿色 `#34D399`
- 开关：关闭 `#3F3F46`（zinc-700），开启 `#34D399`
- 输入框：背景 `#09090B`，边框 `rgba(255,255,255,0.06)`，聚焦时绿色边框

**图表样式（Recharts，统计页）：**
- 柱状图：四色交替填充（绿/紫/黄/粉），hover 亮度提升
- 环形图：使用四色体系
- 折线图：绿色线 + 绿色半透明面积填充
- 坐标轴文字：`rgba(255,255,255,0.25)`，11px
- 网格线：`rgba(255,255,255,0.04)`
- 背景透明

### 4.6 动画规范

| 动画 | 触发 | 参数 |
|------|------|------|
| 数字渐变 | 计时器每秒更新 | 数字平滑递减，无闪烁 |
| 环形进度 | 数据变化 | stroke-dashoffset 过渡, 800ms ease |
| 轨道光点 | 全屏休息时 | 沿圆形轨道匀速旋转, 8s linear infinite |
| 浮窗滑入 | 中提醒触发 | translateX(120%)→0, 300ms ease-out |
| 浮窗滑出 | 关闭/超时 | translateX(0)→translateX(120%), 200ms ease-in |
| 全屏淡入 | 强提醒触发 | opacity 0→1, 500ms ease |
| 页面切换 | 导航点击 | opacity 0→1, 200ms ease |
| 数字滚动 | 统计数字变化 | 从旧值滚动到新值, 400ms |
| 按钮交互 | hover/active | scale(1.02)/scale(0.98), 150ms |
| 暂停过渡 | 暂停/恢复 | 数字透明度 100%↔30%, 300ms ease |

### 4.7 图标

- 使用 `lucide-react` 图标库（细线条风格，和极简排版匹配）
- 导航图标/标签图标：16-20px，默认 `rgba(255,255,255,0.25)`，活跃 `#FAFAFA`
- 设置页分组图标：16px，配合对应颜色
- 状态指示：使用小圆点而非图标

### 4.8 应用图标

- 深色圆角方形底色 `#09090B`
- 中心：四个小圆弧排列成眼睛轮廓，每个弧线使用四色之一（绿/紫/黄/粉）
- 风格：极简线条，和 UI 四色体系呼应

### Phase 1: 项目骨架
```
用 Tauri v2 + React + TypeScript + Tailwind CSS 创建桌面应用项目 EyeGuard。

设计风格：Type + Rings Dark（参考项目 DESIGN.md）
- 主背景 #09090B（近纯黑），卡片背景 #18181B，悬浮态 #27272A
- 四色体系：绿 #34D399（屏幕时间）、紫 #6366F1（休息）、黄 #F59E0B（评分）、粉 #EC4899（警告）
- 主文字 #FAFAFA，次文字 rgba(255,255,255,0.25)
- 字体：全局使用 Inter（Google Fonts），计时器数字 56px weight 800 letter-spacing -3px
- 所有卡片圆角 16px，按钮 8px

功能要求：
1. 系统托盘图标常驻运行，关闭主窗口时最小化到托盘而非退出
2. 托盘左键单击打开主窗口，右键菜单包含：暂停/恢复计时、立即休息、设置、退出
3. 主窗口约 420x580px，内容区域被一个 inset 12px 的圆角边框包围（border-radius 20px，border rgba(255,255,255,0.04)）
4. 顶栏：左侧 "eyeguard" 品牌文字（10px 大写 letter-spacing 3px rgba(255,255,255,0.2)），右侧状态徽章药丸（"active" 绿色半透明底 + 边框）
5. 中央：超大计时器数字 "14:32" 格式，使用 CSS 渐变淡出效果（background: linear-gradient(180deg, #fff 40%, rgba(255,255,255,0.4)); -webkit-background-clip: text; -webkit-text-fill-color: transparent），下方 "minutes remaining" 淡色标签
6. 底部：4 个 SVG 小圆环（60px 直径 3px 描边）横排，各自颜色：绿/紫/黄/粉，环心显示数字（14px 粗体），环下标签（8px 大写），环之间用淡色渐变线连接
7. 顶部标签导航切换页面：timer / stats / pomodoro / settings
```

### Phase 2: 状态检测
```
为 EyeGuard 添加用户活跃状态检测功能：
1. 在 Rust 后端通过全局钩子监听键盘和鼠标事件
2. 有键鼠活动时计时器正常运行，闲置超过 5 分钟自动暂停计时
3. 暂停时：计时器数字透明度降为 30%，状态徽章切换为 "paused"（灰色系）
4. 用户回归后自动恢复，数字恢复正常，徽章切换回 "active"（绿色）
5. 计时结束后触发全屏休息遮罩：
   - 全屏覆盖，背景 rgba(9,9,11,0.94) + backdrop-filter: blur(24px)
   - 中央超大倒计时（Inter 80px font-weight 200 超细体，白色渐变淡出效果，冒号用 rgba(52,211,153,0.6) 绿色点缀）
   - 数字外围一个极淡圆形轨道 rgba(255,255,255,0.03)，轨道上一个缓慢旋转的绿色小光点 #34D399（box-shadow: 0 0 10px，8s linear infinite）
   - 倒计时下方 "Look 20 feet away" 文字（rgba(255,255,255,0.25)）
   - 遮罩淡入 500ms ease
   - 底部 "跳过" ghost 按钮（边框 rgba(255,255,255,0.1)，hover 填充 rgba(255,255,255,0.05)）
6. 闲置阈值 5 分钟为默认值，后续会做成可配置
```

为 EyeGuard 实现分级提醒系统和设置页面，遵循 Type + Rings Dark 设计风格：

     从右侧滑入动画（translateX(120%)→0, 300ms ease-out）
     内容：lucide 图标 + 标题 + 已连续使用时长

### Phase 4: 应用足迹
```
为 EyeGuard 添加应用足迹记录功能：

3. 相邻采样如果是同一进程，合并记录而非新增行
4. 在设置中添加开关可关闭此功能，添加"清除全部数据"按钮
5. 暂时不需要前端展示，下一步做
```
